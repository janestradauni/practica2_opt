Pruebas AMPL organizadas por apartado.

Ejemplos de uso desde la raiz del proyecto:

ampl runs/a/highs.run
ampl runs/a/gurobi.run
ampl runs/a/cplex.run
ampl runs/a/cbc.run
ampl runs/b/highs.run
ampl runs/b/gurobi.run
ampl runs/b/cplex.run
ampl runs/b/cbc.run

Cada fichero .run carga el modelo y los datos desde la raiz del proyecto.

Para NEOS en el apartado B:

- subir `practica2_b.mod`
- subir `practica2.dat`
- subir `runs/b/neos.run`
